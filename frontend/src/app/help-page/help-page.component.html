Main menu

Home

Go to the main page, which shows the copyright message.

Graph

Select Graph to show the graph.

Edit a node and add a child

Right-click a node to amend its attributes. Toggle the Add child to add a new child.

Hide and show a node

Click a node to hide its children. Click it again to show its children.

Delete a node

Click a link to delete a node and its children.

Download

Select Download to save the graph as a JSON file and/or a simplified tree.

Upload

Select Upload to upload a graph saved as a JSON file.
